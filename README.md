LearnersGuild EmojiCipher MellowCoatimundi
==========================================

## Installation and Usage

npm install lg-emoji-mellow-coatimundi
node
> var cipher = require('lg-emoji-mellow-coatimundi')
undefined
> cipher.encode('hello')
💩🤓🖖🖖🍕
> cipher.decode('💦🍕🌂🖖👿')
world
